< Principales etiquetas HTML5 >

HTML5 introdujo varias etiquetas semánticas que proporcionan un significado y estructura más clara al contenido de una página web. Estas etiquetas no solo ayudan a los desarrolladores a escribir código más legible y comprensible, sino que también brindan información adicional a los navegadores y motores de búsqueda sobre la estructura y la relación entre los diferentes elementos en la página. Aquí hay algunas de las principales etiquetas HTML5 semánticas:

1. Elementos iniciales:

  • < !DOCTYPE html >

  • <html> </html>

2. Elementos para metadatos:

  • < head >

  • < title >

  • < link >

  • < meta >

  • < style >

3. Elementos de secciones:

  • < body >

  • < nav >

  • < main >

  • < Section >

  • < Article >

  • < Footer >

  • < header >

4. Elementos semánticos:

  • < a >

  • < strong >

  • < small >

  • < cite >

5. Elementos para creación de tablas:

  • < table >

  • < caption >

  • < colgroup >
